# 🚀 Cosmic AI - Intelligent Automation for the Future

A modern, responsive website showcasing AI services and solutions built with pure HTML, CSS, and JavaScript.

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Clean, professional design with gradient backgrounds and smooth animations
- **6 AI Services** - Machine Learning, NLP, Computer Vision, Data Analytics, AI Security, and Automation
- **6 Key Features** - Expert Team, Custom Solutions, Scalability, 24/7 Support, Proven Results, and Latest Technology
- **Contact Form** - Easy-to-use contact form with email validation
- **Smooth Scrolling** - Enhanced user experience with smooth navigation
- **Zero Dependencies** - Pure HTML, CSS, and JavaScript - no external libraries required

## 🚀 Getting Started

### Option 1: Open Locally
1. Clone or download the repository
2. Open `index.html` in your web browser

### Option 2: Use a Local Server
```bash
# Using Python 3
python -m http.server 8000

# Then visit: http://localhost:8000
```

### Option 3: Deploy to GitHub Pages
1. Go to your repository settings
2. Scroll to "GitHub Pages"
3. Select "main" branch
4. Your site will be live at: `https://elzubir.github.io/cosmic-ai`

## 🎨 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* ... other variables ... */
}
```

### Update Content
All text content can be edited directly in `index.html`.

### Modify Services
Edit or add service cards in the services section of `index.html`:
```html
<div class="service-card">
    <div class="service-icon">🤖</div>
    <h3>Service Name</h3>
    <p>Service description here</p>
</div>
```

### Change Fonts
Modify the `font-family` in `styles.css`:
```css
body {
    font-family: 'Your Font', sans-serif;
}
```
